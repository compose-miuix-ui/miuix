# 颜色系统

本页列出了 Miuix 颜色方案中的所有颜色及其默认的浅色/深色数值。数值为 ARGB 十六进制。

## 使用方式

- 在组合函数中通过 `MiuixTheme.colorScheme.<名称>` 访问。
- 以下展示的颜色适用于非 Monet 模式。
